# 编辑模型应用

[返回上级](../quick_start_mf.md#操作指引)

在“模型工厂>模型应用>新建模型应用”后，您可以通过编辑模型应用信息，从而更新原有模型应用。

在模型应用卡片模式中，单击“操作>编辑”按钮，进入“模型应用编辑”界面。编辑模型应用详情请见“[新建模型应用](./create_model_app.md)”。
